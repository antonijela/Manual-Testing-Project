## ID: 43

## Test Case Name: Verify that the admin can edit a customer's info.

## Pre-Conditions:
- The admin must be logged in (refer to Test Case ID 1 for valid login credentials).

## Steps:
1. Navigate to the left sidebar and click on "Time."
   - Expected result: The Timesheets page is displayed.

2. Click on the "Project Info" option at the top.
   - Expected result: The Project Info dropdown is displayed.

3. Select "Customers" from the dropdown.
   - Expected result: The Customers page is displayed.

4. From the list, next to the desired customer, click the pencil icon under the "Actions" column.
   - Expected result: The Edit Customer page is displayed.

5. Edit the "Name" field. (Input: New Customer Name)
   - Expected result: The customer name is updated successfully.

6. Edit the "Description" field. (Input: Updated Description)
   - Expected result: The description is updated successfully.

7. Click "Save."
   - Expected result: A confirmation message is displayed indicating that the customer's info has been successfully updated.
