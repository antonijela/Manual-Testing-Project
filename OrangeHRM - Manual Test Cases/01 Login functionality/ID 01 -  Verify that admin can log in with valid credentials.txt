## ID: 1

## Test Case Name:
Verify that admin can log in with valid credentials.

## Pre-Conditions:
- The user must have a valid username and password.
- The user is on the login page.

## Steps:
1. Step: Open the login page "https://opensource-demo.orangehrmlive.com/web/index.php/auth/login".
   - Expected result: The login page is displayed.

2. Step: Input valid username in the "Username" field (input: Admin).
   - Expected result: Username is entered and visible.

3. Step: Input valid password in the "Password" field (input: admin123).
   - Expected result: Password is entered, visible, and covered in dots.

4. Step: Click on the "Login" button.
   - Expected result: The user is successfully logged in and is redirected to the dashboard.
